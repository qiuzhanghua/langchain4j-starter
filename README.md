## langchain4j starter

This is a starter project for building applications using the Langchain4j library. It provides a basic structure and examples to help you get started quickly.

### Prerequisites
- Java 21
- Maeven
- Deepseek API Key

### Getting Started
```bash
export DEEPSEEK_API_KEY="your_deepseek_api_key"
```
### Build and Run

```bash
mvn clean package
mvn spring-boot:run
```
